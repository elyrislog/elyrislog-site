# ElyrisLog - Site Institucional (Next.js + Tailwind)

Este é um projeto Next.js pronto para deploy na Vercel.

## Como executar localmente

1. Instale dependências:
```bash
npm install
```
2. Rode em modo dev:
```bash
npm run dev
```

## Deploy na Vercel (passos rápidos)
1. Crie uma conta ou faça login em https://vercel.com.
2. Clique em "New Project" → Import Git Repository (use GitHub/GitLab/Bitbucket).
3. Conecte o repositório com este projeto.
4. Build command: `npm run build` e Output Directory: `.next`.
5. Deploy.

> Observação: eu não tenho permissão para publicar no seu Vercel; siga os passos acima ou me diga se prefere que eu gere um comando `vercel` ou instruções detalhadas para integração com GitHub Actions.
